# AI Prompt · 07 前端页面与组件

## 完整 Prompt

```
【背景】
项目 <name>，前端框架 <Vue/React/...>。本任务生成"页面与组件清单"资产。

【任务】
生成 asset-docs/07-前端-页面与组件清单.md。
按 methodology/03-文档模板与质量标准.md §07 模板。

【输入】
- views/ 或 pages/ 目录
- components/ 目录
- router/ 配置
- 关注：路由映射、API 调用、权限控制、props

【输出字段】
1. 顶层视图（`views/`）表（路由 + 关键 API）
2. 通用组件（`components/Core/`）表（props + 用途）
3. 业务组件（按域分组）表
4. 辅助组件（Tree、Picker、Editor 等）
5. ⚠️ 已知组件问题

【强制列】
- 视图：| 文件 | 路由 | 作用 | 关键 API |
- 组件：| 文件 | 用途 | props |
- 业务组件：| 文件 | 业务 | 关键 API | 权限 |

【关键发现】
- 死代码（未被路由引用）
- 权限字符串错用（复制粘贴）
- 函数名错位
- 改密/校验逻辑漏洞

【反例】
- 不漏死代码
- 不漏权限错用
- 不漏 props 文档

【自我校验】
- 组件数 = `find <src> -name "*.vue" -o -name "*.tsx" | wc -l`
- 死代码数 = `grep -rE "import.*from" <src> | ...` 排除法
- 元信息头 7 字段
```

## 校验清单

- [ ] 组件数与 find 一致
- [ ] 死代码单列
- [ ] 权限错用标注位置
- [ ] 通用组件 props 完整
