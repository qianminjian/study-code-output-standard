## 3. 顶层视图详解

### 3.1 登录页 Login.vue

**文件**：`src/views/login/index.vue`

**关键逻辑**：
```javascript
// 登录表单提交
async handleLogin() {
  this.$refs.form.validate(async (valid) => {
    if (!valid) return;

    // Base64 编码密码
    const formData = {
      username: this.form.username,
      password: btoa(this.form.password),  // ⚠️ 前端 Base64 编码
      authType: 'admin'
    };

    const res = await login(formData);
    if (res.code === 0) {
      this.$store.commit('user/SET_TOKEN', res.data.token);
      this.$router.push('/dashboard');
    }
  });
}
```

## 6. 组件间通信模式

| 模式 | 使用场景 | 实现方式 |
|---|---|---|
| Props Down | 父→子数据传递 | :data="item" |
| Events Up | 子→父事件通知 | @change="handleChange" |
| Vuex/Pinia | 跨组件状态 | this.$store.commit() |
| Provide/Inject | 深嵌套传值 | provide/inject |
| Event Bus | 非父子通信 | Vue.prototype.$bus |

**⚠️ 通信问题识别**：
| 问题 | 位置 | 严重度 | 建议 |
|---|---|---|---|
| Props 链过长 | UserList → UserCard → UserItem | 🟡P2 | 使用 Vuex/Pinia |
| 全局事件滥用 | this.$bus.on/off | 🔴P1 | 使用 Vuex/Pinia |
| 直接修改 Props | :data.sync="item" | 🔴P1 | 改为事件通知 |
