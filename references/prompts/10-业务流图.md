│ [数据层] MySQL: core_account                                      │

## 3. 主业务创建流（流水线创建）

#### 正常路径
```
[用户] 创建流水线
  → POST /pipeline (PipelineController)
  → PipelineService.create()
    → PipelineMapper.insert()
    → StageMapper.batchInsert()
  → 返回 pipelineId
```

#### 异常路径
| 异常场景 | 触发条件 | 当前行为 | 期望行为 | 影响用户 | 优先级 |
|---|---|---|---|:-:|---|
| 流水线名称重复 | 同名已存在 | 未确认处理 | 返回友好错误 | 创建失败 | 🟡 |
| 流水线配置非法 | YAML 格式错误 | 未确认校验 | 返回具体错误位置 | 创建失败 | 🔴 |
| 数据库写入失败 | MySQL 连接断开 | SYSTEM_BUSY | 应有重试 | 创建失败 | 🔴 |
| 权限不足 | 用户无创建权限 | NO_OPERATION_PERMISSION | 同左 | 创建失败 | 🟡 |

## 4. 主业务查询流（资源列表）

#### 正常路径
```
[用户] 访问资源列表页
  → GET /resources (DevopsResourceController)
  → DevopsResourceService.getTree()
    → DevopsResourceMapper.selectTree()
  → 返回用户可见资源树
```

#### 异常路径
| 异常场景 | 触发条件 | 当前行为 | 期望行为 | 影响用户 | 优先级 |
|---|---|---|---|:-:|---|
| Token 过期 | 超过有效期 | TOKEN_EXPIRED | 自动刷新 token | 需重新登录 | 🔴 |
| 项目不存在 | projectId 无效 | 未确认处理 | 返回 404 | 显示异常 | 🟡 |
| 数据权限隔离失效 | 越权访问 | 可能返回空列表 | 应抛出 403 | 数据泄露风险 | 🔴 |
| 注册中心不可用 | Consul/Nacos 宕机 | 可能超时 | 应有降级 | 页面加载慢 | 🟡 |

## 5. 权限校验流

#### 正常路径
```
[用户] 访问受保护资源
  → SSOInterceptor.preHandle()
    → 验证 token 有效性
    → 验证 refreshToken 有效性
    → 解析用户权限
    → 检查访问权限
  → 允许/拒绝请求
```

#### 异常路径
| 异常场景 | 触发条件 | 当前行为 | 期望行为 | 影响用户 | 优先级 |
|---|---|---|---|:-:|---|
| Token 为空 | Header 无 token | TOKEN_REQUIRED | 同左 | 无法访问 | 🔴 |
| Token 格式非法 | 非 JWT 格式 | TOKEN_PARSING_EXCEPTION | 同左 | 无法访问 | 🔴 |
| 权限不足 | 无此操作权限 | NOT_PERMISSION | 同左 | 操作被拒 | 🔴 |
| Session 失效 | session 过期 | SESSION_INVALIDATION | 应自动刷新 | 需重新登录 | 🔴 |

## 7. 改密流

#### 正常路径
```
[用户] 修改密码
  → PUT /password (CoreLoginController)
  → LoginService.changePass()
    → 验证旧密码
    → 更新新密码 (BCrypt)
  → 返回 success
```

#### 异常路径
| 异常场景 | 触发条件 | 当前行为 | 期望行为 | 影响用户 | 优先级 |
|---|---|---|---|:-:|---|
| 旧密码错误 | 输入错误旧密码 | REST_PASSWORD_ERROR | 同左 | 修改失败 | 🔴 |
| 新密码为空 | 新密码为空串 | 未校验 | 应拒绝 | 安全风险 | 🔴 |
| 新旧密码相同 | 输入相同密码 | 未校验 | 应拒绝 | 🟡 |
| 账户不存在 | accountId 无效 | 未确认处理 | 应返回错误 | 修改失败 | 🟡 |
