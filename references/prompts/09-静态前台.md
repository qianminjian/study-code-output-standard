## 5. AJAX 请求封装

### 5.1 全局请求函数

```javascript
// api.js 或 global.js
function request(url, method, data, success, error) {
  $.ajax({
    url: CONFIG.baseUrl + url,
    type: method,
    contentType: 'application/json',
    data: method === 'GET' ? data : JSON.stringify(data),
    headers: {
      'Authorization': 'Bearer ' + localStorage.getItem(CONFIG.tokenKey)
    },
    success: function(res) {
      if (res.code === 0) {
        success && success(res.data);
      } else {
        error && error(res.message);
      }
    },
    error: function(xhr) {
      if (xhr.status === 401) {
        location.href = 'login.html';
      }
      error && error('请求失败');
    }
  });
}

// 简化调用
function get(url, data, callback) {
  request(url, 'GET', data, callback);
}

function post(url, data, callback) {
  request(url, 'POST', data, callback);
}
```

### 5.2 请求封装问题

| 问题 | 位置 | 严重度 | 建议 |
|---|---|---|---|
| 无请求拦截器 | api.js | 🟡P2 | 添加 Loading/错误处理 |
| 无响应拦截器 | api.js | 🟡P2 | 统一 401 处理 |
| 无请求取消 | api.js | 🟡P2 | 防止重复请求 |
| jQuery 依赖 | 全部页面 | 🟡P2 | 考虑原生 fetch |
