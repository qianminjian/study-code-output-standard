# AI Prompt · 09 静态 / 多端

## 完整 Prompt

```
【背景】
项目 <name>，静态站技术 <jQuery/...>，或多端 <iOS/Android/小程序>。本任务生成"静态/多端"资产。

【任务】
生成 asset-docs/09-<静态端名>.md。
按 references/methodology/03-文档模板与质量标准.md §09 模板。

【输入】
- 静态站根目录 / 移动端项目根目录
- HTML 入口 / 移动端入口
- 全局 JS / CSS

【输出字段】
1. 文件结构
2. 页面清单（HTML / View / Activity / Page）
3. 关键 JS / 模式（AJAX / 渲染 / 跳转）
4. ⚠️ 已知问题

【关键发现】
- baseUrl 硬编码（内网 IP / 端口）
- 富文本渲染无 XSS 过滤（`.html(content)`）
- 跨子项目配置不同步（端口 / 路径）
- 引用旧版 jQuery / 库

【反例】
- 不漏 XSS 风险
- 不漏 baseUrl 硬编码
- 不漏跨子项目配置差异

【自我校验】
- HTML 数 = `find <static-dir> -name "*.htm" -o -name "*.html" | wc -l`
- JS 数 = `find <js-dir> -name "*.js" | wc -l`
- 元信息头 7 字段
```
