## 5. 继承层级（BaseModel 分析）

```
BaseModel (抽象基类)
├── id: Long (主键)
├── createBy: String
├── createTime: LocalDateTime
├── updateBy: String
├── updateTime: LocalDateTime
└── deleted: Integer (逻辑删除)

   ▲ 继承
   │
UserModel extends BaseModel
├── username: String
├── password: String (加密)
└── status: Integer
```

| 基类 | 子类 | 继承字段 |
|---|---|---|
| BaseModel | User, Role, Permission | id, createTime, updateTime, deleted |
| BaseEntity | BusinessLog | + bizType, bizId |

|---|---|---|---|

## 8. 索引与性能

| 表 | 索引名 | 字段 | 类型 | 用途 |
|---|---|---|---|---|
| sys_user | idx_username | username | 唯一索引 | 登录查询 |
| sys_user | idx_status | status | 普通索引 | 状态筛选 |

**缺失索引识别**：
- 外键字段无索引 → 🔴P1 性能风险
- 频繁 WHERE 字段无索引 → 🟡P2
